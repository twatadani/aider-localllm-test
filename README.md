# aider-localllm-test
Aiderを介したローカルLLMのテスト

## ローカルLLMに作らせるタスクのリスト

すべてCLI専用、Pythonで作る。

### 初級編

1. 階乗計算  
コマンドライン引数でnを受け取り、nの階乗を計算してターミナルに表示する。  

2. フィボナッチ数列
コマンドライン引数でnを受け取り、フィボナッチ数列(n)を計算してターミナルに表示する。

3. 文字列逆順
引数の文字列を逆順に表示する

4. GCD計算
コマンドライン引数を2つ受け取り、ユークリッド互除法でそれらの最大公約数を表示する。その際、ユークリッド互除法をCLI上でグラフィカルに表示する。

5. 引数合計
コマンドライン引数を複数受け取り、それらの合計と平均を表示する。

### 中級編

6. 行/単語カウント
コマンドライン引数でファイルパスを受け取り、ファイルの行数、単語数、文字数を表示する。

7. パスワード生成
文字種、長さ指定でセキュアなパスワードの候補を生成

8. CLI電卓
コマンドライン引数を評価する関数電卓

9. プログレスバー
n秒カウントダウン中にターミナルにプログレスバーを表示。ETAも表示。

### 上級編

10. ASCIIアート変換
コマンドライン引数の入力テキストをASCIIアートにする

11. ディレクトリツリー
コマンドライン引数でパスを受け取り、それ以下のディレクトリツリーとそのディスクサイズをターミナル上でグラフィカルに表示する。

12. ソート視覚化
ランダムな数字のリストを生成し、バブルソートする。ステップごとにターミナルに過程を表示する。

13. 五目並べ
ユーザーと対話的に五目並べを行う。
